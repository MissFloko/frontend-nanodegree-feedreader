# Feed Reader Testing Project<br>Udacity Front-End Web Devloper Nanodegree

This project is using the JavaScript testing framework Jasmine on a feed reader website.

## How to access the test

- OPTION 1
    * clone the repository on your cumputer using you terminal<br>
        `git clone https://missfloko.github.io/frontend-nanodegree-feedreader/`
    * open the folder
    * open the index.html in your favorite browser
- OPTION 2
    * download the zip folder from the repository on GitHub
    * open the folder
    * open the index.html in your favorite browser
- OPTION 3
    * fork the repository on GitHub on your computer
    * open the folder
    * open the index.html in your favorite browser
- OPTION 4
    * visit [this link](https://missfloko.github.io/frontend-nanodegree-feedreader/)
